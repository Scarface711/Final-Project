# チーム開発演習(Git/GitHub)　最終課題　レベル1　提出
## 概要
本サイト「Table Guide」は、架空のレストランを題材にしたレビューサイト制作の成果物（デモ）です。生成した写真風ビジュアル（外観・内観・料理・ワイン）と、店舗概要文・メニュー紹介文・利用規約／プライバシーポリシー／お問い合わせ／FAQ／ヘルプ等の固定ページを組み合わせ、実在のサービスを想定した情報設計と画面構成を確認できるようにしています。掲載している店舗（例：BRILLARE、Lumina、桜亭、Parvati、Campo dorato ほか）はすべて架空であり、画像・名称・本文はデモ目的で作成したものです。レイアウトはBootstrapを用いたレスポンシブ設計を前提とし、店舗一覧・ジャンル導線・カード表示・リンク構造など、レビューサイトとして必要なUI要素を一通り揃えています。
